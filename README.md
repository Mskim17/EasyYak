# 약 봉투 글씨 읽어드려요 (Node.js)

약 봉투(또는 약 안내 종이) 사진을 올리면 글씨를 읽어서(OCR) 보여주고, 어려운 용어는 쉬운 우리말로 바꿔서 출력하는 간단한 웹앱입니다.

## 실행 방법

```bash
npm install
npm start
```

그다음 브라우저에서 `http://localhost:3000` 을 여세요.

## 사용 방법

- 휴대폰에서 실행하면 **사진 고르기 → 읽어보기**만 누르면 됩니다.
- 결과는
  - **중요한 내용**: 복용 횟수/시간/갯수처럼 핵심만 추려서 보여줌
  - **전체 글씨 보기**: 쉬운 말로 바꾼 전체 문장
  - **원래 글씨(그대로)**: OCR 원문

## 쉬운 말로 바꾸기 로직

`server.js`의 `EASY_KOREAN_DICT`(사전+규칙)로 치환합니다. 현장에서 자주 쓰는 표현이 있으면 항목을 더 추가하면 됩니다.

## 참고

- OCR은 `tesseract.js`를 사용합니다. 사진이 어둡거나 흔들리면 정확도가 떨어질 수 있어요. 밝고 정면에서 찍을수록 좋습니다.
